## 💡 About the Project

This portfolio is a Single-Page Application (SPA) crafted to provide a seamless overview of my engineering journey. It emphasizes:

* **Professionalism:** A clean, intuitive layout for quick recruiter review.
* **Technical Rigor:** Direct links to my LeetCode and GeeksforGeeks progress, showcasing consistent problem-solving.
* **Modern Stack:** Built for performance and responsiveness across all devices.

## 🛠 Tech Stack

**Frontend:** React.js, Tailwind CSS

**Deployment:** Netlify

**Version Control:** Git & GitHub

## 🚀 Key Features

* **Algorithmic Tracking:** Integrated display of my competitive programming status (LeetCode Guardian Badge / GeeksforGeeks Knight Badge).
* **Skills Visualization:** Dynamic representation of proficiency across Frontend, Backend, AI/ML, and Database technologies.
* **Direct Access:** One-click access to my resume and key professional profiles (LinkedIn, GitHub, Email).
* **Responsive Layout:** A fluid, mobile-first design built with Tailwind CSS.
